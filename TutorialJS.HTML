1. Apa itu JavaScript?

JavaScript adalah bahasa pemrograman yang membuat halaman web menjadi hidup dan interaktif. Jika HTML adalah kerangka (struktur) dan CSS adalah kulit (tampilan), maka JavaScript adalah otak dan gerakan (logika & perilaku).

Contoh: Klik tombol di bawah, teks akan berubah.

JavaScript bisa merespon klik, mengubah konten, mengirim data, dan bahkan membuat animasi.

2. Cara Menyematkan JavaScript

  • Internal JS → ditulis dalam tag <script> di dalam HTML (biasanya di akhir body).
  • External JS → file terpisah .js dihubungkan dengan <script src="file.js">.
  • Inline JS → ditulis langsung di atribut elemen (contoh: onclick).
<script> console.log("Hello dari internal"); </script> <script src="script.js"></script> <button onclick="alert('Inline')">Klik</button>

💡 Rekomendasi: Letakkan <script> sebelum </body> agar HTML terlebih dahulu dimuat.

3. Output JavaScript

  • innerHTML → mengubah isi elemen HTML.
  • alert() → popup peringatan.
  • console.log() → untuk debugging (buka console browser F12).

4. Variabel: let, const, var

Variabel adalah tempat menyimpan data.

  • let → nilai bisa diubah (scope blok). Rekomendasi utama.
  • const → nilai tetap (konstanta), tidak bisa diubah.
  • var → cara lama (hindari).
let umur = 16; umur = 17; // diperbolehkan const nama = "Andi"; // nama = "Budi"; // ❌ Error

5. Tipe Data

  • String → teks, contoh: "Halo"
  • Number → angka, contoh: 10, 3.14
  • Boolean → true/false
  • Array → kumpulan data, contoh: [1,2,3]
  • Object → pasangan key-value, contoh: {nama:"Andi", umur:20}

6. Operator Dasar

  • Aritmatika: + - * / %
  • Perbandingan: ==, ===, !=, >, <
  • Logika: && (AND), || (OR), ! (NOT)

7. Percabangan (if else)

Untuk mengambil keputusan berdasarkan kondisi.

let nilai = 80; if (nilai >= 75) { console.log("Lulus"); } else { console.log("Tidak lulus"); }

8. Perulangan (Loop)

for dan while digunakan untuk mengulang kode.

for (let i = 0; i < 5; i++) { console.log(i); }

9. Fungsi (Function)

Blok kode yang bisa dipanggil berulang kali.

function sapa(nama) { return "Halo " + nama; } let pesan = sapa("Andi");

10. DOM Manipulation (Mengubah HTML)

Dengan JavaScript, kita bisa mengubah konten, style, dan atribut elemen HTML.

  • document.getElementById("id") → mengambil elemen
  • element.innerHTML = "teks" → mengubah isi
  • element.style.color = "red" → mengubah style

Teks ini akan berubah

11. Event Handler (onclick, onmouseover)

Event adalah aksi yang terjadi di halaman (klik, hover, input).

12. Array dan Object

let buah = ["Apel", "Mangga", "Jeruk"]; let siswa = { nama: "Budi", umur: 17 };

13. LocalStorage (Penyimpanan Browser)

Data bisa disimpan di browser pengguna meskipun halaman di-refresh.

14. Kesalahan Umum & Tips

  • Lupa mendeklarasikan variabel dengan let/const.
  • Menggunakan = (assignment) bukan == atau ===.
  • Memanggil fungsi sebelum didefinisikan (hoisting hanya berlaku untuk deklarasi fungsi).
  • Buka console browser (F12) untuk melihat error.

💡 Tips: Gunakan console.log() untuk debug, dan biasakan memberi nama variabel yang jelas.
