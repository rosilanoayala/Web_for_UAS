📘 C++ Tutorial

1. Pengantar C++ (Introduction)

C++ adalah bahasa pemrograman yang sangat populer dan powerful. Diciptakan oleh Bjarne Stroustrup pada tahun 1985 sebagai pengembangan dari bahasa C.

💡 Analogi Sederhana

Bayangkan Anda ingin membangun rumah. Bahasa tingkat tinggi (seperti Python) ibarat Anda membeli rumah jadi (mudah tapi tidak bisa atur pondasi). Bahasa tingkat rendah (Assembly) ibarat Anda membuat batu bata sendiri dari pasir (sangat detail dan sulit). Nah, C++ ada di tengah-tengah: Anda bisa mengatur struktur dengan leluasa, tapi banyak bahan bangunan yang sudah disediakan.

🔍 Bagaimana C++ Bekerja?

Berbeda dengan bahasa web yang langsung dibaca oleh browser, program C++ harus melalui proses yang disebut kompilasi. Ini adalah proses menerjemahkan kode yang Anda tulis menjadi "bahasa mesin" (kode biner 0 dan 1) agar bisa dijalankan oleh komputer.

// 1. Anda menulis kode ini #include <iostream> using namespace std; int main() { cout << "Hello, World!"; return 0; } // 2. Compiler akan memprosesnya menjadi file program (.exe) // 3. Komputer akan mengeksekusi (menjalankan) file program tersebut

🧱 Struktur Dasar Program (Kerangka Wajib)

Setiap kali Anda membuat program C++, kerangka dasarnya akan selalu terlihat seperti ini:

#include <iostream> // 1. Memanggil buku panduan (library) bernama iostream using namespace std; // 2. Jalan pintas agar tidak perlu menulis std:: terus int main() { // 3. Ini adalah ruang mesin utama program Anda cout << "Hello"; // 4. Semua instruksi ditulis di dalam kurung kurawal ini return 0; // 5. Lapor ke komputer bahwa program berjalan lancar }

🚀 Kenapa Belajar C++?

  • Kecepatan Maksimal: Karena langsung diubah ke bahasa mesin, C++ adalah salah satu bahasa tercepat di dunia.
  • Membuat Sistem Besar: Windows, macOS, Photoshop, dan mesin pencari Google raksasa dibangun dengan bantuan C++.
  • Raja Game Engine: Jika Anda ingin membuat game 3D dengan Unreal Engine, C++ adalah bahasa wajibnya.

⚠️ Hati-hati Jika Kamu Pemula!

C++ ibarat mobil balap. Sangat cepat, namun tidak punya "rem otomatis". Kesalahan kecil (seperti lupa tanda titik koma ;) bisa membuat program gagal berjalan. Tetap bersabar dan baca pesan error-nya ya!

2. Struktur dan Syntax Dasar C++

Setiap program C++ memiliki struktur baku yang harus diikuti. Jika struktur dasar ini salah, compiler akan menolak program Anda dengan memunculkan pesan error.

💡 Analogi Struktur

Sama seperti menulis surat resmi yang butuh Kop Surat, Salam Pembuka, Isi Surat, dan Penutup, program C++ juga butuh "surat pengantar" agar dimengerti oleh komputer.

🔍 Anatomi Kode: Penjelasan Per Baris

Mari kita bedah program sederhana ini baris per baris:

#include <iostream> // Baris 1: Meminta izin pakai library input/output using namespace std; // Baris 2: Memakai kamus standar (std) int main() { // Baris 3: Titik awal program berjalan cout << "Halo"; // Baris 4: Menampilkan teks ke layar return 0; // Baris 5: Selesai tanpa error } // Baris 6: Tutup program
  • #include <iostream>: Menambahkan fitur cout (menampilkan teks) dan cin (meminta input) ke program Anda.
  • using namespace std;: Agar Anda cukup menulis cout, tidak perlu repot-repot menulis panjang menjadi std::cout setiap saat.
  • int main(): Fungsi utama. Wajib ada! Tanpa main, program tidak akan tahu dari mana harus mulai berjalan.
  • return 0;: Kode rahasia ke Windows/Linux bahwa program berhasil dijalankan tanpa crash.

⚠️ Tiga Kesalahan Utama Pemula

  1. Lupa titik koma (;). C++ butuh ; di setiap akhir perintah.
  2. Salah Huruf Besar/Kecil. C++ sangat sensitif! Cout (salah) berbeda dengan cout (benar).
  3. Kurung tidak seimbang. Setiap buka { harus ada tutup }.

3. Variabel: Menyimpan Data Anda

Variabel adalah "kotak" tempat Anda menyimpan data di memori komputer. Karena C++ sangat peduli pada kerapian, setiap kotak (variabel) harus memiliki satu label nama dan tipe barang yang spesifik.

💡 Analogi Variabel

Anggaplah program Anda adalah dapur. Anda butuh toples bulat untuk gula (angka), dan butuh kulkas panjang untuk sayur (teks). Variabel bekerja persis seperti ini: menyedikan tempat yang pas untuk setiap jenis data.

🧱 Cara Membuat Variabel

Di C++, Anda harus mendeklarasikan variabel sebelum menggunakannya. Anda memberitahu komputer: *"Tolong siapkan kotak kecil bersisi angka, beri nama 'umur'"*.

int umur = 20; // integer: bilangan bulat double tinggi = 170.5; // double: bilangan desimal char kelas = 'A'; // char: 1 karakter tunggal string nama = "Budi"; // string: kalimat (butuh teks kutip dua) bool lulus = true; // boolean: true atau false

🔄 Variabel Bisa Diubah Nilainya

Sesuai namanya (variabel), isinya bisa diubah-ubah di tengah program berjalan tanpa perlu menyebut tipe datanya lagi.

int skor = 0; // Skor awal skor = 10; // Skor sekarang menjadi 10 skor = skor + 5; // Skor bertambah 5 (menjadi 15) cout << skor; // Output akan menampilkan angka 15

⚠️ Jangan Lupakan Ini!

1. Nama variabel **case-sensitive** (Skor tidak sama dengan skor).
2. Jangan buat variabel diawali dengan angka (seperti 1nama).
3. Selalu beri nilai awal! Variabel kosong (misal cuma tulis int skor;) bisa dipenuhi data sampah rahasia peninggalan komputer yang membuat hasil error.

🧪 Coba Program Penuh Ini!

Di bawah ini adalah contoh utuh menggunakan variabel string dan int. Tekan tombol "Coba kode ini" di bawah untuk bereksperimen langsung.

#include <iostream> using namespace std; int main() { string nama = "Andi"; int usia = 20; cout << "Halo " << nama << "!" << endl; cout << "Tahun ini usia Anda " << usia << " tahun." << endl; cout << "Tahun depan, usia Anda jadi " << usia + 1 << " tahun." << endl; return 0; }

4. Tipe Data (Data Types)

Setelah kita belajar tentang variabel, sekarang saatnya memahami tipe data. Tipe data adalah jenis nilai yang dapat disimpan dalam sebuah variabel. Bayangkan Anda memiliki beberapa kotak penyimpanan: ada kotak khusus untuk menyimpan angka bulat, kotak untuk angka desimal, kotak untuk huruf, dan kotak untuk teks panjang. Setiap kotak memiliki fungsi dan kapasitas yang berbeda. Di C++, kita harus menentukan tipe data variabel saat mendeklarasikannya, agar compiler tahu berapa banyak memori yang perlu disediakan dan operasi apa yang diizinkan.

Memilih tipe data yang tepat sangat penting karena:

  • Menghemat penggunaan memori (misalnya, menyimpan nilai kecil di tipe yang lebih besar akan boros).
  • Menghindari kesalahan perhitungan (misalnya, menyimpan desimal di tipe integer akan membulatkan angka).
  • Membuat program lebih mudah dibaca dan dirawat.

📊 Jenis-Jenis Tipe Data Dasar di C++

Berikut adalah tipe data yang paling sering digunakan oleh pemula. Setelah Anda menguasai ini, nanti bisa mempelajari tipe data lain seperti short, long, long long, unsigned, dll.

  • int → bilangan bulat (integer), contoh: 10, -5, 0, 1000. Ukuran biasanya 4 byte (bisa menampung nilai dari sekitar -2 miliar hingga 2 miliar).
  • float → bilangan desimal (floating point) dengan presisi rendah (sekitar 6-7 digit desimal). Contoh: 3.14, -0.001, 2.5. Ukuran 4 byte.
  • double → bilangan desimal dengan presisi tinggi (sekitar 15-16 digit desimal). Contoh: 3.14159265358979. Ukuran 8 byte. Rekomendasi: lebih sering gunakan double daripada float untuk akurasi.
  • char → satu karakter, ditulis dalam kutip tunggal. Contoh: 'A', '5', '&', ' '. Ukuran 1 byte.
  • string → teks (kumpulan karakter), butuh #include <string>. Contoh: "Andi", "Halo dunia!". Ukuran bervariasi.
  • bool → nilai logika, hanya dua kemungkinan: true (benar) atau false (salah). Biasanya digunakan untuk kondisi dalam percabangan atau perulangan.

🧪 Contoh Deklarasi dan Inisialisasi

int umur = 17; // bilangan bulat double tinggi = 170.5; // desimal presisi tinggi float berat = 55.75f; // desimal presisi rendah (beri akhiran f) char grade = 'A'; // satu karakter string nama = "Budi"; // teks bool aktif = true; // true atau false

Perhatikan bahwa untuk float, kita bisa menambahkan huruf f di akhir angka untuk memberi tahu compiler bahwa ini adalah float (bukan double).

⚠️ Kesalahan Umum yang Sering Terjadi

// ❌ SALAH: int tidak bisa menyimpan desimal int nilai = 3.5; // nilai akan menjadi 3 (bagian desimal hilang) // ❌ SALAH: char hanya untuk satu karakter, bukan string char huruf = "A"; // error! pakai kutip dua untuk string // ✅ BENAR: int nilai = 3; // integer char huruf = 'A'; // char pakai kutip satu string teks = "A"; // string pakai kutip dua

Kesalahan lain: lupa menyertakan #include <string> saat menggunakan tipe string. Jika tidak, compiler akan error karena tidak mengenal string.

🔍 Memori dan Ukuran (Sedikit Lebih Dalam)

Setiap tipe data memiliki ukuran memori yang tetap (kecuali string yang dinamis). Anda bisa mengecek ukuran dengan operator sizeof():

#include <iostream> using namespace std; int main() { cout << "Ukuran int: " << sizeof(int) << " byte" << endl; cout << "Ukuran double: " << sizeof(double) << " byte" << endl; cout << "Ukuran char: " << sizeof(char) << " byte" << endl; cout << "Ukuran bool: " << sizeof(bool) << " byte" << endl; return 0; } // Output (bisa berbeda tergantung sistem): // Ukuran int: 4 byte // Ukuran double: 8 byte // Ukuran char: 1 byte // Ukuran bool: 1 byte

Mengetahui ukuran ini berguna saat Anda bekerja dengan memori terbatas (misalnya pada mikrokontroler atau game dengan performa tinggi).

🎯 Kapan Menggunakan Tipe Data Tertentu?

  • int → untuk hitungan, umur, indeks, jumlah barang (tidak butuh desimal).
  • double → untuk pengukuran, perhitungan ilmiah, uang (butuh akurasi desimal).
  • char → untuk memproses karakter satu per satu (misal dalam parsing teks).
  • string → untuk menyimpan nama, alamat, kalimat.
  • bool → untuk flag (penanda), status on/off, hasil perbandingan.

💡 Tips Penting untuk Pemula

  • Jika ragu antara float dan double, pilih double. Lebih aman dan akurat, meskipun memakan memori dua kali lipat (tapi biasanya tidak masalah untuk program kecil).
  • Jangan gunakan int untuk menyimpan nilai desimal, karena akan terjadi truncation (pemotongan) yang tidak Anda sadari.
  • Selalu inisialisasi variabel saat mendeklarasikannya. Variabel yang tidak diinisialisasi berisi nilai "sampah" (nilai acak dari memori) dan bisa menyebabkan bug misterius.
  • Gunakan nama variabel yang mencerminkan tipe data (misal: jumlahMahasiswa untuk int, rataRataNilai untuk double).

🧪 Contoh Program Lengkap dengan Berbagai Tipe Data

#include <iostream> #include <string> using namespace std; int main() { string nama; int usia; double ipk; bool lulus; cout << "Masukkan nama: "; getline(cin, nama); cout << "Masukkan usia: "; cin >> usia; cout << "Masukkan IPK: "; cin >> ipk; lulus = (ipk >= 2.75); cout << "\n--- Data Mahasiswa ---" << endl; cout << "Nama : " << nama << endl; cout << "Usia : " << usia << " tahun" << endl; cout << "IPK : " << ipk << endl; cout << "Status lulus: " << (lulus ? "Ya" : "Tidak") << endl; return 0; }

Program di atas menggunakan string, int, double, dan bool untuk menyimpan data mahasiswa. Perhatikan bagaimana bool digunakan untuk menyimpan hasil perbandingan (ipk >= 2.75).

🚀 Kesimpulan

Tipe data adalah fondasi dari setiap variabel. Memahami perbedaan antara int, double, char, string, dan bool akan membantu Anda menulis program yang lebih efisien dan bebas bug. Di section selanjutnya, kita akan belajar tentang input dari pengguna (cin) dan operator aritmatika. Pastikan Anda sudah paham tipe data sebelum melanjutkan.

5. Input dari Pengguna (User Input)

Selama ini kita hanya menampilkan teks ke layar menggunakan cout. Namun program yang baik biasanya berinteraksi dengan pengguna: menerima data, memprosesnya, lalu menampilkan hasil. Input dari pengguna membuat program menjadi dinamis dan personal. Di C++, kita menggunakan objek cin (character input) untuk membaca data dari keyboard.

🔍 Dasar-dasar cin

Perhatikan contoh sederhana berikut:

int umur; cin >> umur;

Penjelasan:

  • cin adalah objek bawaan C++ untuk membaca input dari keyboard.
  • Operator >> (ekstraksi) mengambil data dari cin dan menyimpannya ke variabel di sebelah kanan.
  • Program akan berhenti sejenak dan menunggu pengguna mengetik sesuatu lalu menekan Enter.
  • Input harus sesuai dengan tipe data variabel. Jika variabel int dan pengguna mengetik huruf, program bisa error atau berperilaku tidak terduga.

🧪 Contoh Program Lengkap dengan Input

#include <iostream> using namespace std; int main() { int umur; cout << "Masukkan umur Anda: "; cin >> umur; cout << "Umur Anda adalah " << umur << " tahun." << endl; return 0; }

Jika pengguna memasukkan angka 20, outputnya: Umur Anda adalah 20 tahun.

📝 Membaca String dengan cin (Hati-hati dengan Spasi)

cin menggunakan spasi, tab, atau newline sebagai pemisah. Artinya, jika pengguna mengetik Andi Saputra, hanya "Andi" yang akan terbaca, sisanya tertinggal di buffer. Perhatikan contoh:

string nama; cout << "Masukkan nama lengkap: "; cin >> nama; // hanya membaca kata pertama cout << "Halo " << nama << endl; // Input: Andi Saputra // Output: Halo Andi

Untuk membaca kalimat lengkap yang mengandung spasi, kita perlu fungsi getline().

📚 Membaca String dengan getline()

Fungsi getline(cin, variabel) membaca seluruh baris hingga pengguna menekan Enter, termasuk spasi.

string namaLengkap; cout << "Masukkan nama lengkap: "; getline(cin, namaLengkap); cout << "Halo " << namaLengkap << endl; // Input: Andi Saputra // Output: Halo Andi Saputra

⚠️ Masalah Umum: cin dan getline Bersamaan

Ini adalah jebakan klasik yang sering membuat pemula frustasi. Perhatikan kode berikut:

int umur; string nama; cout << "Masukkan umur: "; cin >> umur; cout << "Masukkan nama: "; getline(cin, nama); // kenapa langsung terlewati? cout << "Umur: " << umur << ", Nama: " << nama << endl;

Setelah cin >> umur, pengguna menekan Enter. Karakter newline ('\n') yang dihasilkan tetap tertinggal di buffer input. Kemudian getline(cin, nama) membaca newline tersebut sebagai baris kosong, sehingga seolah-olah nama tidak sempat diisi.

🛠️ Solusi: Membersihkan Buffer dengan cin.ignore()

Untuk mengatasi masalah di atas, kita perlu membersihkan buffer sebelum getline. Gunakan cin.ignore() yang akan membuang satu karakter (biasanya newline) dari buffer.

int umur; string nama; cout << "Masukkan umur: "; cin >> umur; cin.ignore(); // membuang newline yang tertinggal cout << "Masukkan nama: "; getline(cin, nama); cout << "Umur: " << umur << ", Nama: " << nama << endl;

Jika Anda perlu membersihkan lebih banyak karakter (misal setelah membaca angka, ada banyak spasi), gunakan cin.ignore(numeric_limits<streamsize>::max(), '\n') untuk membersihkan seluruh baris hingga newline. Tapi untuk pemula, cin.ignore() satu kali sudah cukup.

🧠 Alur Kerja Input (Visualisasi)

Program berjalan → menampilkan prompt → pengguna mengetik → tekan Enter → data masuk ke buffer → cin atau getline mengambil data → data disimpan ke variabel → program lanjut

💡 Tips Penting untuk Pemula

  • Selalu beri pesan yang jelas sebelum meminta input, misal cout << "Masukkan nilai: "; agar pengguna tahu apa yang harus diketik.
  • Jika program Anda menggunakan cin dan getline secara berurutan, jangan lupa cin.ignore() di antara keduanya.
  • Hindari mencampur cin untuk angka dan getline untuk string tanpa membersihkan buffer.
  • Untuk membaca satu kata tanpa spasi, gunakan cin. Untuk membaca kalimat lengkap, gunakan getline.
  • Selalu validasi input jika memungkinkan (misal periksa apakah angka yang dimasukkan positif).

🧪 Contoh Program Interaktif dengan Validasi Sederhana

#include <iostream> #include <string> using namespace std; int main() { string nama; int nilai; cout << "=== Program Nilai Mahasiswa ===" << endl; cout << "Masukkan nama: "; getline(cin, nama); cout << "Masukkan nilai (0-100): "; cin >> nilai; if (nilai >= 60) { cout << "Selamat " << nama << ", Anda LULUS!" << endl; } else { cout << "Maaf " << nama << ", Anda TIDAK LULUS." << endl; } return 0; }

Program di atas menggunakan getline untuk nama (bisa mengandung spasi) dan cin untuk angka. Karena tidak ada cin sebelum getline, tidak perlu cin.ignore(). Urutan input sangat penting.

🚀 Kesimpulan

Input dari pengguna membuat program Anda hidup. Anda telah mempelajari cara menggunakan cin untuk angka dan satu kata, serta getline untuk kalimat lengkap. Jangan lupa atasi masalah buffer dengan cin.ignore() saat menggabungkan kedua metode. Di section selanjutnya, kita akan membahas operator aritmatika untuk melakukan perhitungan berdasarkan input pengguna.

6. Operator dalam C++

Operator adalah simbol khusus yang memberitahu compiler untuk melakukan operasi tertentu terhadap data. Tanpa operator, kita tidak bisa menghitung, membandingkan, atau menggabungkan nilai. Di C++, operator dikelompokkan menjadi beberapa jenis: aritmatika, perbandingan, logika, dan assignment.

➕ Operator Aritmatika (Matematika Dasar)

Operator ini digunakan untuk perhitungan matematika seperti yang biasa kita lakukan di sekolah.

  • + (penjumlahan) → 5 + 3 menghasilkan 8
  • - (pengurangan) → 5 - 3 menghasilkan 2
  • * (perkalian) → 5 * 3 menghasilkan 15
  • / (pembagian) → 5 / 2 menghasilkan 2 (karena integer division, pecahan dibuang)
  • % (modulus / sisa bagi) → 5 % 2 menghasilkan 1 (sisa dari 5 dibagi 2)
int a = 10, b = 3; int tambah = a + b; // 13 int kurang = a - b; // 7 int kali = a * b; // 30 int bagi = a / b; // 3 (bukan 3.333) int sisa = a % b; // 1

Peringatan penting: Jika kedua operand adalah integer, maka / melakukan pembagian integer (hasil dibulatkan ke bawah). Jika ingin hasil desimal, pastikan salah satu operand bertipe double atau float.

⚖️ Operator Perbandingan (Membandingkan Nilai)

Operator perbandingan digunakan untuk membandingkan dua nilai. Hasilnya selalu berupa bool (true atau false). Operator ini sangat penting dalam percabangan (if).

  • == (sama dengan) → 5 == 5 true, 5 == 3 false
  • != (tidak sama dengan) → 5 != 3 true
  • > (lebih besar) → 5 > 3 true
  • < (lebih kecil) → 5 < 3 false
  • >= (lebih besar atau sama) → 5 >= 5 true
  • <= (lebih kecil atau sama) → 5 <= 5 true
int x = 10, y = 20; bool hasil1 = (x == y); // false bool hasil2 = (x != y); // true bool hasil3 = (x > y); // false bool hasil4 = (x < y); // true cout << hasil1 << hasil2 << hasil3 << hasil4; // 0 1 0 1

Catatan: Di C++, true direpresentasikan sebagai 1, dan false sebagai 0 saat dicetak.

🔗 Operator Logika (Menggabungkan Kondisi)

Operator logika digunakan untuk menggabungkan dua atau lebih kondisi. Hasilnya juga bool. Sering dipakai dalam if yang kompleks.

  • && (logika AND) → benar jika kedua kondisi benar.
  • || (logika OR) → benar jika salah satu kondisi benar.
  • ! (logika NOT) → membalik nilai kebenaran (true jadi false, false jadi true).
int usia = 20; bool punyaSIM = true; bool bisaMengemudi = (usia >= 17 && punyaSIM); // true bool bolehLiburan = (usia >= 17 || punyaSIM); // true bool tidakBisa = !bisaMengemudi; // false

📝 Operator Assignment (Memberi Nilai)

Operator assignment digunakan untuk mengisi nilai ke variabel. Yang paling dasar adalah =, tetapi ada juga operator gabungan yang mempersingkat penulisan.

  • = → x = 5; (x menjadi 5)
  • += → x += 3; sama dengan x = x + 3;
  • -= → x -= 2; sama dengan x = x - 2;
  • *= → x *= 4; sama dengan x = x * 4;
  • /= → x /= 2; sama dengan x = x / 2;
  • %= → x %= 3; sama dengan x = x % 3;
int x = 10; x += 5; // x sekarang 15 x -= 3; // x sekarang 12 x *= 2; // x sekarang 24 x /= 4; // x sekarang 6 x %= 5; // x sekarang 1

⚠️ Kesalahan Klasik: Assignment (=) vs Perbandingan (==)

Ini adalah kesalahan paling sering dilakukan pemula. Satu tanda sama dengan = adalah assignment (memberi nilai), sedangkan dua tanda sama dengan == adalah perbandingan (apakah sama?).

int x = 5; if (x = 10) { // ❌ SALAH: ini akan memberi nilai 10 ke x, kondisi selalu true cout << "x adalah 10"; } if (x == 10) { // ✅ BENAR: membandingkan apakah x sama dengan 10 cout << "x adalah 10"; }

Compiler modern biasanya memberi peringatan, tapi program tetap berjalan dengan logika yang salah.

🧮 Prioritas Operator (Urutan Eksekusi)

Sama seperti matematika, operator dalam C++ memiliki urutan prioritas. Perkalian dan pembagian didahulukan daripada penjumlahan dan pengurangan. Jika ingin mengubah urutan, gunakan tanda kurung ( ).

int hasil1 = 5 + 2 * 3; // 5 + (2*3) = 11 int hasil2 = (5 + 2) * 3; // (7)*3 = 21

Prioritas lengkap (dari tertinggi ke terendah): () → * / % → + - → < <= > >= → == != → && → || → =.

💡 Tips Penting

  • Gunakan tanda kurung untuk memperjelas maksud Anda, meskipun tidak diperlukan. Contoh: hasil = (a + b) * c; lebih mudah dibaca.
  • Operator % hanya bisa digunakan untuk integer (tidak untuk double/float).
  • Hati-hati dengan pembagian integer: 5/2 menghasilkan 2, bukan 2.5. Jika ingin hasil desimal, gunakan 5.0/2 atau 5/2.0.
  • Operator logika && dan || menggunakan short-circuit evaluation: jika kondisi pertama sudah menentukan hasil, kondisi kedua tidak dievaluasi. Contoh: (false && apapun) hasilnya false tanpa mengecek apapun.

7. Percabangan (Conditional Statements)

Percabangan adalah kemampuan program untuk mengambil keputusan berdasarkan suatu kondisi. Dengan percabangan, program tidak berjalan lurus, tetapi bisa memilih tindakan yang berbeda sesuai situasi. Ini seperti "jika-maka" dalam kehidupan sehari-hari.

Analogi sederhana: Jika hujan, maka bawa payung; jika tidak, tidak perlu payung. Program juga bisa seperti itu.

🔹 Struktur Dasar if-else

Bentuk paling sederhana dari percabangan adalah if (jika) dan else (jika tidak).

if (kondisi) { // kode ini dijalankan jika kondisi bernilai true } else { // kode ini dijalankan jika kondisi bernilai false }

Contoh: Cek apakah seseorang sudah cukup umur untuk memilih.

int umur = 17; if (umur >= 17) { cout << "Anda sudah bisa memiliki KTP."; } else { cout << "Anda masih terlalu muda."; }

🔹 Percabangan dengan Banyak Pilihan (else if)

Jika terdapat lebih dari dua kemungkinan, gunakan else if. Program akan mengecek kondisi dari atas ke bawah dan menjalankan blok pertama yang benar, lalu keluar dari struktur if.

int nilai = 85; if (nilai >= 90) { cout << "A"; } else if (nilai >= 75) { cout << "B"; } else if (nilai >= 60) { cout << "C"; } else { cout << "D"; }

Urutan kondisi sangat penting. Jika kita tulis if (nilai >= 75) sebelum if (nilai >= 90), maka nilai 95 akan dianggap "B", bukan "A".

⚠️ Kesalahan Umum yang Sering Terjadi

  • Lupa kurung kurawal { } → Tanpa kurung, hanya satu baris setelah if/else yang dianggap bagian dari blok. Selalu gunakan kurung meskipun hanya satu baris.
  • Menggunakan = bukan == → if (x = 5) akan memberi nilai 5 ke x dan kondisi selalu benar (kecuali nilai 0). Gunakan if (x == 5) untuk perbandingan.
  • Urutan kondisi salah → Letakkan kondisi paling spesifik di atas (misal nilai >= 90) sebelum yang lebih umum (nilai >= 75).
  • Lupa titik koma setelah kondisi? Jangan! → if (x > 5); (ada titik koma) akan mengakhiri if tanpa melakukan apapun.

🔹 Percabangan Bersarang (Nested if)

Kita bisa menempatkan if di dalam if. Ini berguna untuk kondisi yang lebih kompleks.

int usia = 20; bool punyaSIM = true; if (usia >= 17) { if (punyaSIM) { cout << "Anda boleh mengemudi."; } else { cout << "Anda cukup umur tapi belum punya SIM."; } } else { cout << "Anda belum cukup umur."; }

💡 Tips Penting

  • Gunakan else if untuk rantai kondisi yang saling eksklusif.
  • Jika hanya ada satu kondisi, gunakan if saja tanpa else.
  • Untuk memeriksa apakah suatu nilai berada dalam rentang, gunakan if (x >= 10 && x <= 20).
  • Kode akan lebih mudah dibaca jika setiap blok diindentasi dengan rapi.

Dengan percabangan, program Anda menjadi "cerdas" karena bisa menyesuaikan perilaku berdasarkan data.

8. Perulangan (Loop)

Perulangan digunakan untuk menjalankan kode yang sama berulang kali tanpa harus menulis ulang. Bayangkan Anda ingin mencetak "Halo" 100 kali. Tanpa perulangan, Anda harus menulis 100 baris cout. Dengan perulangan, cukup 3 baris. Ada dua jenis perulangan utama di C++: while dan for.

🔹 While Loop

while digunakan ketika jumlah perulangan tidak diketahui sebelumnya. Program akan terus mengulang selama kondisi bernilai true.

int i = 0; while (i < 5) { cout << i << " "; i++; } // Output: 0 1 2 3 4

Penjelasan:

  • Variabel i diinisialisasi dengan 0.
  • Kondisi i < 5 dicek. Jika true, masuk ke dalam loop.
  • Di dalam loop, nilai i dicetak, lalu i++ (menambah 1).
  • Setelah mencapai i = 5, kondisi menjadi false, loop berhenti.

⚠️ Peringatan Infinite Loop: Jika Anda lupa menambah nilai i, maka kondisi akan selalu true dan loop tidak pernah berhenti. Program akan hang atau crash. Selalu pastikan ada perubahan yang mengarah ke kondisi berhenti.

// INFINITE LOOP (JANGAN DIJALANKAN) int i = 0; while (i < 5) { cout << i; // lupa i++ }

🔹 For Loop

for digunakan ketika jumlah perulangan sudah diketahui. Semua komponen loop (inisialisasi, kondisi, increment) ditulis dalam satu baris.

for (int i = 0; i < 5; i++) { cout << i << " "; } // Output: 0 1 2 3 4

Struktur for:

  • int i = 0 → inisialisasi (dijalankan sekali di awal).
  • i < 5 → kondisi (diperiksa setiap sebelum iterasi).
  • i++ → increment (dijalankan setelah setiap iterasi).

For loop sangat cocok untuk memproses array atau melakukan iterasi dengan indeks.

🔹 Kapan Menggunakan While vs For?

  • while → cocok jika jumlah iterasi tidak pasti, misal: "ulangi sampai pengguna memasukkan angka positif".
  • for → cocok jika jumlah iterasi sudah diketahui, misal: "cetak angka 0 sampai 99".
// Contoh while untuk validasi input int angka; cout << "Masukkan angka positif: "; cin >> angka; while (angka <= 0) { cout << "Input tidak valid. Masukkan angka positif: "; cin >> angka; }

🔹 Perulangan Tak Terbatas (Infinite Loop) yang Disengaja

Kadang kita sengaja membuat loop tak terbatas, misalnya pada program server atau game yang terus berjalan sampai ada perintah berhenti. Untuk menghentikannya, gunakan break di dalam loop.

while (true) { cout << "Program berjalan... "; // jika ada kondisi berhenti, gunakan break; }

💡 Tips dan Analogi

  • While seperti: "Selama masih ada beras, masak nasi".
  • For seperti: "Masak nasi sebanyak 10 kali".
  • Selalu pastikan loop memiliki kondisi berhenti (jika tidak, akan infinite).
  • Gunakan break untuk keluar paksa dari loop, dan continue untuk melompat ke iterasi berikutnya.

9. Array (Larik)

Bayangkan Anda ingin menyimpan nilai ujian 100 siswa. Apakah Anda akan membuat 100 variabel terpisah seperti nilai1, nilai2, ..., nilai100? Tentu tidak. Array adalah solusi: sebuah wadah yang dapat menyimpan banyak nilai dengan tipe data yang sama, dan diakses menggunakan indeks.

🔹 Mendeklarasikan dan Menginisialisasi Array

int angka[5]; // array 5 elemen (belum diisi) int nilai[3] = {10, 20, 30}; // langsung diisi

Setelah deklarasi, setiap elemen array memiliki nomor indeks dimulai dari 0. Jadi:

  • nilai[0] = 10
  • nilai[1] = 20
  • nilai[2] = 30

Penting: Indeks terakhir adalah ukuran-1. Jika array berukuran 5, indeks valid adalah 0,1,2,3,4.

🔹 Mengakses dan Mengubah Nilai Array

int arr[3] = {5, 10, 15}; cout << arr[0]; // mencetak 5 arr[1] = 99; // mengubah elemen indeks 1 menjadi 99 cout << arr[1]; // 99

⚠️ Kesalahan Umum: Out of Bounds

Jika Anda mengakses indeks di luar batas (misal arr[5] untuk array ukuran 5), program bisa crash atau menghasilkan nilai acak. Compiler C++ tidak memeriksa batas array secara otomatis, jadi tanggung jawab programmer untuk memastikan indeks valid.

int arr[3] = {1,2,3}; cout << arr[3]; // ERROR! indeks 3 tidak ada (maks 2)

🔹 Menggunakan Loop untuk Memproses Array

Loop (terutama for) sangat cocok untuk mengakses semua elemen array.

int nilai[5] = {80, 85, 90, 75, 88}; int total = 0; for (int i = 0; i < 5; i++) { total += nilai[i]; } double rata = total / 5.0; cout << "Rata-rata: " << rata << endl;

🔹 Array dan Ukuran Konstan

Ukuran array harus ditentukan saat kompilasi (konstanta). Namun kita bisa menggunakan const int agar mudah diubah di satu tempat.

const int JUMLAH_SISWA = 5; int nilai[JUMLAH_SISWA]; for (int i = 0; i < JUMLAH_SISWA; i++) { cout << "Masukkan nilai siswa ke-" << i+1 << ": "; cin >> nilai[i]; }

💡 Tips dan Trik

  • Selalu pastikan indeks array tidak melebihi ukuran - 1.
  • Jika array diinisialisasi dengan nilai, compiler bisa menentukan ukuran secara otomatis: int arr[] = {1,2,3}; (ukuran 3).
  • Array tidak bisa di-assign atau dibandingkan secara langsung (gunakan loop).
  • Untuk array karakter (string gaya C), ada aturan khusus, tapi sebaiknya gunakan string untuk teks.

Array adalah fondasi untuk struktur data yang lebih kompleks seperti matriks (array 2D), vector, dan seterusnya. Kuasai array dengan baik karena akan sering digunakan.

10. Functions (Fungsi)

Bayangkan Anda sering menulis kode yang sama berulang kali, misalnya untuk menghitung luas persegi panjang. Daripada menulis rumus yang sama di sepuluh tempat, Anda bisa membuat fungsi. Fungsi adalah blok kode yang diberi nama dan dapat dipanggil kapan saja. Ini membuat program lebih rapi, mudah dirawat, dan tidak boros baris kode.

🔹 Mengapa Fungsi Sangat Penting?

  • Reusability (Dapat Digunakan Ulang) → Tulis sekali, panggil berkali-kali.
  • Modularitas → Program dipecah menjadi bagian-bagian kecil yang independen.
  • Mudah Debug → Cukup perbaiki satu fungsi, semua pemanggilan otomatis terperbaiki.
  • Abstraksi → Kita tidak perlu tahu detail cara kerja fungsi, cukup tahu cara menggunakannya.

🔹 Anatomi Fungsi

tipeData namaFungsi(daftarParameter) { // badan fungsi (body) return nilai; // jika tipeData bukan void }
  • tipeData → jenis nilai yang dikembalikan (int, double, string, void, dll). Jika tidak mengembalikan apa pun, gunakan void.
  • namaFungsi → harus deskriptif (misal hitungLuas, cetakPesan).
  • daftarParameter → variabel input yang diterima fungsi (bisa kosong).
  • return → mengirimkan nilai kembali ke pemanggil. Fungsi void tidak perlu return (bisa pakai return; untuk keluar lebih awal).

🔹 Contoh Fungsi Sederhana (Tanpa Parameter, Tanpa Return)

void tampilkanMenu() { cout << "=== MENU UTAMA ===" << endl; cout << "1. Mulai Game" << endl; cout << "2. Pengaturan" << endl; cout << "3. Keluar" << endl; } int main() { tampilkanMenu(); // panggil fungsi return 0; }

Fungsi tampilkanMenu hanya mencetak teks, tidak mengembalikan nilai, dan tidak butuh input. Setiap kali Anda butuh menu, cukup panggil fungsi ini.

🔹 Fungsi dengan Parameter (Input)

Parameter memungkinkan fungsi bekerja dengan data yang berbeda setiap kali dipanggil.

void sapaUser(string nama) { cout << "Halo, " << nama << "! Selamat belajar C++." << endl; } int main() { sapaUser("Andi"); sapaUser("Budi"); return 0; } // Output: // Halo, Andi! Selamat belajar C++. // Halo, Budi! Selamat belajar C++.

Parameter nama akan diisi dengan nilai yang diberikan saat pemanggilan.

🔹 Fungsi yang Mengembalikan Nilai (Return)

Fungsi bisa mengolah input dan menghasilkan output. Contoh: fungsi penjumlahan.

int tambah(int a, int b) { return a + b; } int main() { int hasil = tambah(5, 3); cout << "5 + 3 = " << hasil; // 8 return 0; }

Setelah return, fungsi langsung berhenti. Kode setelah return tidak akan dijalankan.

🔹 Fungsi dengan Banyak Parameter

double hitungLuasPersegiPanjang(double panjang, double lebar) { return panjang * lebar; } int main() { double luas = hitungLuasPersegiPanjang(5.5, 3.2); cout << "Luas: " << luas << endl; return 0; }

Parameter dipisah dengan koma. Urutan parameter harus sesuai saat memanggil.

⚠️ Kesalahan Umum Pemula

  • Lupa menulis tipe data return → Misal menulis tambah(int a, int b) { return a+b; } tanpa int di depan. Compiler akan error.
  • Lupa menulis return pada fungsi non-void → Nilai yang dikembalikan tidak terdefinisi (bisa nilai acak).
  • Memanggil fungsi sebelum dideklarasikan → Solusi: deklarasikan fungsi di atas main(), atau gunakan function prototype di atas.
  • Tipe data parameter tidak sesuai → Misal fungsi minta int tapi diberi double (bisa terjadi konversi implisit, tapi hati-hati).
// Contoh function prototype int tambah(int a, int b); // deklarasi int main() { cout << tambah(5,3); } int tambah(int a, int b) { // definisi return a + b; }

💡 Tips Penting

  • Nama fungsi harus verb (kata kerja) atau menggambarkan aksi: hitungRata(), cetakStruk(), ambilInput().
  • Fungsi yang terlalu panjang (lebih dari 30 baris) sebaiknya dipecah menjadi fungsi-fungsi kecil.
  • Gunakan const pada parameter jika fungsi tidak perlu mengubah nilai input.
  • Fungsi void bisa menggunakan return; untuk keluar lebih awal (misal jika kondisi error).

Dengan fungsi, Anda bisa membangun program yang kompleks dari potongan-potongan kecil yang mudah dipahami.

11. Strings (Teks)

Dalam pemrograman, kita sering perlu menyimpan dan memanipulasi teks, seperti nama, alamat, atau kalimat. Di C++, tipe data untuk teks adalah string. String adalah kumpulan karakter (huruf, angka, simbol) yang dirangkai menjadi satu kesatuan.

🔹 Menggunakan String di C++

Untuk menggunakan string, Anda harus menyertakan library <string> dan menggunakan namespace std.

#include <iostream> #include <string> using namespace std; int main() { string nama = "Andi"; cout << nama << endl; return 0; }

🔹 Membaca Input String dengan Spasi (getline)

Jika menggunakan cin >> nama, hanya kata pertama yang terbaca (berhenti di spasi). Untuk membaca seluruh baris (termasuk spasi), gunakan getline(cin, variabel).

string alamat; cout << "Masukkan alamat lengkap: "; getline(cin, alamat); cout << "Alamat: " << alamat << endl;

Catatan: Jika sebelumnya Anda menggunakan cin untuk angka, jangan lupa membersihkan buffer dengan cin.ignore() sebelum getline.

🔹 Operasi Dasar pada String

  • Panjang string → nama.length() atau nama.size() (sama saja).
  • Menggabungkan (concatenation) → menggunakan operator +.
  • Akses karakter → seperti array: nama[indeks] (indeks mulai 0).
  • Membandingkan → ==, !=, <, > (berdasarkan urutan leksikografis).
string depan = "Andi"; string belakang = " Saputra"; string lengkap = depan + belakang; // "Andi Saputra" cout << "Panjang: " << depan.length() << endl; // 4 cout << "Huruf pertama: " << depan[0] << endl; // A depan[0] = 'B'; cout << depan; // "Bndi"

🔹 Perulangan pada String

Kita bisa memproses setiap karakter dalam string menggunakan loop for.

string kata = "Belajar"; for (int i = 0; i < kata.length(); i++) { cout << "Karakter ke-" << i << ": " << kata[i] << endl; }

🔹 Membandingkan Dua String

string a = "Andi"; string b = "Andi"; if (a == b) { cout << "Sama" << endl; } else { cout << "Berbeda" << endl; }

Perbandingan bersifat case-sensitive: "Andi" tidak sama dengan "andi".

⚠️ Masalah Umum: cin dan getline Bersamaan

Setelah cin >> angka, newline (\n) tertinggal di buffer. Kemudian getline membaca newline itu sebagai baris kosong. Solusi: panggil cin.ignore() untuk membuang newline.

int umur; string nama; cout << "Umur: "; cin >> umur; cin.ignore(); // bersihkan buffer cout << "Nama: "; getline(cin, nama); cout << "Nama: " << nama << ", Umur: " << umur;

🔹 Method String Lainnya yang Berguna

  • substr(pos, panjang) → mengambil substring.
  • find("teks") → mencari posisi teks dalam string.
  • empty() → mengecek apakah string kosong.
  • append("teks") → menambah teks di akhir (mirip +).
string s = "Halo dunia"; cout << s.substr(0, 4); // "Halo" cout << s.find("dunia"); // 5 (posisi mulai) if (s.empty()) cout << "kosong";

💡 Tips

  • Gunakan getline untuk membaca input yang mungkin mengandung spasi.
  • Selalu #include <string> jika menggunakan tipe string.
  • Hindari cin dan getline bergantian tanpa cin.ignore().
  • String di C++ bisa diubah-ubah (mutable), berbeda dengan string di beberapa bahasa lain yang immutable.

12. Boolean (Nilai Logika)

Dalam pemrograman, sering kita perlu menyimpan nilai yang hanya memiliki dua kemungkinan: benar atau salah. Misalnya: apakah lampu menyala? apakah pengguna sudah login? apakah nilai di atas batas? Di C++, tipe data untuk ini adalah bool (singkatan dari boolean).

🔹 Nilai Boolean

bool hanya bisa bernilai true (benar) atau false (salah). Dalam memori, true direpresentasikan sebagai 1, false sebagai 0.

bool isLogin = true; bool isActive = false; cout << isLogin; // output: 1 cout << isActive; // output: 0

🔹 Mengapa Boolean Penting?

  • Dasar dari percabangan (if, else) dan perulangan (while, for).
  • Menyimpan status atau kondisi yang bersifat ya/tidak.
  • Membuat kode lebih mudah dibaca (misal bool isRaining lebih jelas daripada int rainingFlag).

🔹 Ekspresi Perbandingan Menghasilkan Boolean

Operator perbandingan seperti ==, !=, >, < menghasilkan nilai boolean.

int nilai = 80; bool lulus = (nilai >= 75); // true if (lulus) { cout << "Selamat, Anda lulus!"; }

🔹 Operator Logika pada Boolean

Kita bisa menggabungkan beberapa kondisi menggunakan operator logika:

  • && (AND) → benar jika kedua kondisi benar.
  • || (OR) → benar jika salah satu kondisi benar.
  • ! (NOT) → membalik nilai (true jadi false, false jadi true).
int usia = 20; bool punyaSIM = true; bool bisaMengemudi = (usia >= 17 && punyaSIM); // true bool bolehMasuk = (usia >= 18 || punyaSIM); // true bool tidakBisa = !bisaMengemudi; // false

🔹 Penggunaan Boolean dalam Loop

bool lanjut = true; int angka; while (lanjut) { cout << "Masukkan angka (0 untuk berhenti): "; cin >> angka; if (angka == 0) { lanjut = false; } else { cout << "Anda memasukkan " << angka << endl; } }

🔹 Konversi ke Boolean

Di C++, nilai non-nol dianggap true, nilai nol dianggap false. Ini bisa berguna tapi juga bisa menjebak.

int x = 10; if (x) { // sama dengan if (x != 0) cout << "x bukan nol"; }

Namun, untuk kejelasan, sebaiknya selalu tulis kondisi secara eksplisit: if (x != 0).

⚠️ Kesalahan Umum

  • Menulis if (isLogin = true) (assignment) bukan if (isLogin == true) atau cukup if (isLogin).
  • Lupa bahwa && dan || memiliki short-circuit: jika kondisi pertama sudah menentukan hasil, kondisi kedua tidak dievaluasi.
// Short-circuit: jika ptr == nullptr, maka ptr->value tidak akan dievaluasi if (ptr != nullptr && ptr->value > 0) { ... }

💡 Tips

  • Gunakan nama variabel boolean yang diawali kata kerja bantu: isReady, hasFinished, canProceed.
  • Jangan bandingkan boolean dengan true/false secara eksplisit: if (isReady) lebih baik daripada if (isReady == true).
  • Boolean sangat berguna sebagai flag untuk mengontrol alur program.

Dengan memahami boolean, Anda siap menggunakan logika kompleks dalam program.

13. Switch (Percabangan dengan Banyak Pilihan)

Ketika Anda memiliki satu variabel yang perlu dibandingkan dengan banyak nilai tetap (misalnya kode menu 1,2,3 atau huruf A,B,C), menggunakan if-else akan menghasilkan banyak baris kode yang berulang. Switch adalah alternatif yang lebih rapi dan mudah dibaca untuk situasi seperti ini.

🔹 Struktur Dasar Switch

switch (ekspresi) { case nilai1: // kode jika ekspresi == nilai1 break; case nilai2: // kode jika ekspresi == nilai2 break; // ... bisa banyak case default: // kode jika tidak ada case yang cocok }
  • ekspresi → biasanya variabel (bertipe int, char, atau enum).
  • case → nilai konstanta yang dibandingkan dengan ekspresi.
  • break → menghentikan eksekusi switch, mencegah "jatuh" ke case berikutnya.
  • default → opsional, dijalankan jika tidak ada case yang cocok.

📘 Contoh Praktis: Menentukan Nama Hari

int hari = 2; switch (hari) { case 1: cout << "Senin"; break; case 2: cout << "Selasa"; break; case 3: cout << "Rabu"; break; case 4: cout << "Kamis"; break; case 5: cout << "Jumat"; break; case 6: cout << "Sabtu"; break; case 7: cout << "Minggu"; break; default: cout << "Hari tidak valid"; } // Output: Selasa

🔹 Mengapa Perlu break? (Fall-Through)

Jika Anda tidak menulis break, program akan terus menjalankan kode pada case berikutnya, meskipun nilainya tidak cocok. Ini disebut fall-through. Kadang fall-through disengaja (misalnya untuk beberapa case yang ingin menjalankan kode yang sama), tetapi sering menyebabkan bug.

// Tanpa break (HATI-HATI) int angka = 1; switch (angka) { case 1: cout << "Satu "; case 2: cout << "Dua "; } // Output: Satu Dua (karena tidak ada break)

Contoh fall-through yang disengaja (berguna):

char grade = 'B'; switch (grade) { case 'A': cout << "Luar biasa! "; case 'B': cout << "Bagus! "; case 'C': cout << "Cukup. "; break; default: cout << "Perlu belajar lagi."; } // Jika grade 'A', output: Luar biasa! Bagus! Cukup. // Jika grade 'B', output: Bagus! Cukup. // Jika grade 'C', output: Cukup.

⚠️ Hal yang Perlu Diperhatikan

  • Nilai pada case harus berupa konstanta (angka, karakter, atau enum), tidak bisa variabel atau ekspresi.
  • Switch hanya bisa digunakan untuk tipe data integral: int, char, enum, dan short. Tidak bisa untuk string atau float/double.
  • Jangan lupa menulis break pada setiap case, kecuali memang sengaja ingin fall-through.
  • Case tidak boleh duplikat.
  • Default boleh diletakkan di mana saja (biasanya di akhir), tetapi jika di tengah, tetap butuh break.

💡 Kapan Menggunakan Switch vs if-else?

  • Gunakan switch jika Anda membandingkan satu variabel dengan banyak nilai konstanta yang tetap (seperti menu, kode error, status).
  • Gunakan if-else jika kondisi melibatkan rentang (misal nilai >= 90), operator logika (&&, ||), atau tipe data non-integral (string, float).

Switch membuat kode lebih bersih dan berpotensi lebih cepat (compiler dapat mengoptimalkan). Namun, jangan memaksakan jika kondisi kompleks.

14. Break dan Continue (Mengontrol Alur Perulangan)

Dalam perulangan (for, while), kadang kita perlu keluar lebih awal atau melewati satu iterasi. Di sinilah break dan continue berguna.

🔹 Break: Menghentikan Paksa Loop

break akan menghentikan loop sepenuhnya, meskipun kondisi loop masih true. Program akan melanjutkan ke baris setelah loop.

for (int i = 0; i < 5; i++) { if (i == 3) { break; // berhenti saat i = 3 } cout << i << " "; } // Output: 0 1 2

Contoh nyata: Mencari sebuah angka dalam array. Begitu ketemu, tidak perlu melanjutkan pencarian.

int cari = 25; int arr[] = {10, 20, 30, 25, 40}; int found = -1; for (int i = 0; i < 5; i++) { if (arr[i] == cari) { found = i; break; // ketemu, keluar loop } } if (found != -1) cout << "Ditemukan di indeks " << found; else cout << "Tidak ditemukan";

🔹 Continue: Lompati Satu Iterasi

continue akan melewati sisa kode dalam iterasi saat ini dan langsung melompat ke iterasi berikutnya (dengan mengevaluasi kondisi loop lagi).

for (int i = 0; i < 5; i++) { if (i == 3) { continue; // lewati i = 3 } cout << i << " "; } // Output: 0 1 2 4

Contoh nyata: Mencetak semua bilangan genap dari 1 sampai 10, melewati yang ganjil.

for (int i = 1; i <= 10; i++) { if (i % 2 != 0) continue; // jika ganjil, lewati cout << i << " "; } // Output: 2 4 6 8 10

⚠️ Peringatan Penting

  • break dan continue hanya bekerja di dalam loop (for, while, do-while) atau switch (khusus break).
  • Jangan gunakan break di luar loop/switch – akan menyebabkan error kompilasi.
  • Dalam loop bersarang (loop di dalam loop), break hanya menghentikan loop terdalam tempat ia berada.
  • Gunakan continue dengan hati-hati: pastikan tidak menyebabkan infinite loop (misal lupa increment).

💡 Analogi Kehidupan

  • Break → Anda sedang mengantri, tiba-tiba Anda memutuskan keluar dari antrian (tidak kembali lagi).
  • Continue → Dalam antrian, Anda membiarkan satu orang di depan Anda dilayani lebih dulu (Anda mundur satu langkah), lalu Anda tetap dalam antrian.

Dengan break dan continue, Anda bisa mengontrol loop dengan lebih presisi, membuat program lebih efisien dan ekspresif.

15. Struct (Mengelompokkan Data)

Dalam pemrograman, sering kita berhadapan dengan data yang saling terkait, misalnya data mahasiswa: nama, NIM, umur, IPK. Daripada membuat empat variabel terpisah, kita bisa menggabungkannya menjadi satu kesatuan menggunakan struct. Struct adalah tipe data bentukan yang dapat berisi beberapa anggota (member) dengan tipe data berbeda.

🔹 Mendefinisikan Struct

struct Mahasiswa { string nama; int nim; double ipk; };

Kode di atas mendefinisikan tipe data baru bernama Mahasiswa yang memiliki tiga anggota: nama (string), nim (int), ipk (double).

🔹 Membuat Variabel Struct

Mahasiswa mhs1; // deklarasi variabel mhs1.nama = "Andi"; mhs1.nim = 12345; mhs1.ipk = 3.75; cout << mhs1.nama << " (NIM: " << mhs1.nim << ") memiliki IPK " << mhs1.ipk;

Anggota struct diakses menggunakan operator titik ..

🔹 Inisialisasi Struct (C++11 ke atas)

Mahasiswa mhs2 = {"Budi", 67890, 3.90}; // urutan sesuai deklarasi // atau Mahasiswa mhs3{"Citra", 11223, 3.85};

🔹 Struct dengan Array di Dalamnya

struct MataKuliah { string nama; int sks; int nilai[3]; // array nilai tugas, UTS, UAS };

🔹 Struct Bersarang (Nested Struct)

Struct dapat berisi struct lain.

struct Alamat { string jalan; string kota; string provinsi; }; struct Mahasiswa { string nama; int nim; Alamat alamat; // struct di dalam struct }; Mahasiswa mhs; mhs.alamat.kota = "Jakarta";

🔹 Array of Struct

Kita bisa membuat array yang setiap elemennya adalah struct.

Mahasiswa daftar[3] = { {"Andi", 101, 3.5}, {"Budi", 102, 3.7}, {"Citra", 103, 3.9} }; for (int i = 0; i < 3; i++) { cout << daftar[i].nama << " IPK: " << daftar[i].ipk << endl; }

🔹 Struct sebagai Parameter Fungsi

Anda bisa mengirim struct ke fungsi (bisa dengan pass-by-value atau pass-by-reference).

void cetakMahasiswa(Mahasiswa m) { cout << m.nama << " - " << m.nim << endl; } // panggil: cetakMahasiswa(mhs1);

🔹 Mengapa Struct Penting?

  • Mengorganisasi data → data yang berkaitan dikelompokkan.
  • Memudahkan pengelolaan → cukup satu variabel untuk mewakili sekumpulan data.
  • Dasar OOP → class di C++ pada dasarnya adalah struct dengan tambahan fungsi dan kontrol akses.

💡 Tips dan Perbedaan dengan Class

  • Secara teknis, di C++, struct dan class hampir sama, kecuali: anggota struct bersifat public secara default, sedangkan class private.
  • Untuk kumpulan data sederhana (hanya data, tidak ada method), gunakan struct. Untuk objek yang memiliki perilaku (method), gunakan class.
  • Biasakan memberi nama struct dengan huruf kapital di awal.
  • Struct sangat berguna saat Anda perlu mengembalikan beberapa nilai dari fungsi (return sebuah struct).

Dengan struct, Anda bisa merepresentasikan data dunia nyata dengan lebih alami dan kode menjadi lebih terstruktur.

16. References (Alias Variabel)

Reference adalah nama lain (alias) untuk variabel yang sudah ada. Bayangkan Anda memiliki seorang teman yang memiliki nama panggilan. Nama panggilan itu merujuk ke orang yang sama. Begitu pula reference: ia memberikan nama alternatif ke suatu variabel, tanpa menyalin datanya.

🔹 Membuat Reference

string nama = "Andi"; string &ref = nama; // ref adalah reference ke nama cout << ref; // Output: Andi ref = "Budi"; // mengubah nilai melalui reference cout << nama; // Output: Budi (nilai asli berubah)

Perhatikan simbol & di tipe data. Ini menandakan bahwa ref bukan variabel biasa, melainkan reference.

🔹 Mengapa Reference Berguna?

  • Menghindari penyalinan data besar → Saat Anda mengirim variabel ke fungsi, secara default nilai disalin (pass-by-value). Jika variabel berukuran besar (misal array atau struct besar), penyalinan memakan waktu dan memori. Dengan reference, tidak ada penyalinan, fungsi langsung mengakses data asli.
  • Memungkinkan fungsi mengubah variabel asli → Dengan pass-by-reference, fungsi bisa memodifikasi argumen asli, mirip seperti pointer tetapi dengan sintaks yang lebih sederhana.
  • Lebih aman daripada pointer → Reference tidak bisa NULL dan tidak bisa diubah ke variabel lain setelah diinisialisasi.

🔹 Reference sebagai Parameter Fungsi

void tambahSatu(int &x) { x++; // langsung mengubah variabel asli } int main() { int a = 5; tambahSatu(a); cout << a; // Output: 6 return 0; }

Tanpa reference, tambahSatu(a) hanya akan mengubah salinan lokal, dan a tetap 5.

🔹 Reference dengan const (Keamanan Ekstra)

Jika Anda ingin mengirim data besar tanpa menyalin, tetapi tidak ingin fungsi mengubah data asli, gunakan const reference.

void cetakData(const string &data) { cout << data; // hanya baca, tidak bisa mengubah // data = "baru"; // ERROR! karena const }

Ini sangat umum digunakan untuk efisiensi.

⚠️ Perbedaan Reference dan Pointer

  • Reference harus diinisialisasi saat deklarasi; pointer boleh tidak diinisialisasi (tapi berbahaya).
  • Reference tidak bisa diubah untuk menunjuk ke variabel lain setelah dibuat; pointer bisa diubah kapan saja.
  • Reference tidak bisa NULL; pointer bisa NULL (dan harus diperiksa sebelum digunakan).
  • Mengakses nilai reference seperti variabel biasa; pointer perlu dereference (*ptr).

💡 Tips Penggunaan Reference

  • Gunakan reference untuk parameter fungsi yang perlu mengubah argumen asli (output parameter).
  • Gunakan const reference untuk parameter yang hanya perlu dibaca tetapi berukuran besar (string, struct, array).
  • Jangan mengembalikan reference ke variabel lokal fungsi (akan menjadi dangling reference).
  • Reference lebih disarankan daripada pointer jika memungkinkan, karena lebih aman dan mudah digunakan.

Reference adalah fitur C++ yang membuat kode lebih efisien dan bersih. Pahami perbedaannya dengan pointer, dan Anda akan menggunakannya setiap hari.

17. Pointers (Penunjuk Alamat Memori)

Jika reference adalah "nama lain", maka pointer adalah "alamat rumah". Pointer menyimpan alamat memori dari suatu variabel. Dengan pointer, Anda bisa mengakses dan memanipulasi data di alamat tersebut. Ini memberikan kontrol tingkat rendah yang sangat powerful, tetapi juga rawan kesalahan.

🔹 Memori dan Alamat (Konsep Dasar)

Setiap variabel disimpan di memori RAM pada alamat tertentu (bilangan heksadesimal). Anda bisa melihat alamat tersebut menggunakan operator &.

int angka = 10; cout << &angka; // misal output: 0x7ffd8a9b (alamat heksa)

🔹 Mendeklarasikan Pointer

Pointer dideklarasikan dengan menambahkan tanda bintang * setelah tipe data.

int *ptr; // pointer ke integer double *dPtr; // pointer ke double string *sPtr; // pointer ke string

🔹 Menyimpan Alamat ke Pointer

int angka = 10; int *ptr = &angka; // ptr menyimpan alamat angka

🔹 Mengakses Nilai melalui Pointer (Dereference)

Gunakan operator * (bintang) lagi untuk mendapatkan nilai yang disimpan di alamat yang ditunjuk.

cout << *ptr; // mencetak 10 *ptr = 20; // mengubah nilai asli melalui pointer cout << angka; // 20

🔹 Pointer dan Array

Nama array sebenarnya adalah pointer ke elemen pertama. Ini hubungan yang sangat erat.

int arr[3] = {10, 20, 30}; int *ptr = arr; // sama dengan &arr[0] cout << *ptr; // 10 cout << *(ptr+1); // 20 (ptr+1 melompat ke elemen berikutnya) // Perhatikan: pointer arithmetic menambah sesuai ukuran tipe data (int = 4 byte)

🔹 Pointer NULL (Pointer Kosong)

Pointer bisa diinisialisasi dengan nullptr (C++11) atau NULL (C++98) untuk menandakan bahwa ia tidak menunjuk ke mana pun. Jangan pernah dereference pointer NULL (akan crash).

int *ptr = nullptr; if (ptr != nullptr) { cout << *ptr; // aman }

🔹 Alokasi Memori Dinamis (new dan delete)

Salah satu kegunaan utama pointer adalah membuat objek di heap (memori dinamis) yang hidupnya diatur oleh programmer.

int *ptr = new int(5); // alokasi integer di heap, nilai awal 5 cout << *ptr; // 5 *ptr = 10; delete ptr; // dealokasi, hindari memory leak ptr = nullptr; // kebiasaan baik

Setiap new harus dipasangkan dengan delete. Jika lupa, program akan bocor memori.

⚠️ Kesalahan Umum dan Bahaya Pointer

  • Dereference pointer NULL → program crash.
  • Memory leak → lupa delete setelah new.
  • Dangling pointer → pointer menunjuk ke memori yang sudah didealokasi.
  • Wild pointer → pointer belum diinisialisasi, menunjuk ke alamat acak.

💡 Kapan Menggunakan Pointer?

  • Membuat struktur data dinamis (linked list, tree, graph).
  • Alokasi memori dengan ukuran yang tidak diketahui saat kompilasi (misal array dinamis).
  • Untuk efisiensi saat mengirim data besar (tapi lebih disarankan reference jika memungkinkan).
  • Berinteraksi dengan fungsi C atau hardware.

Pointer adalah topik yang sulit bagi pemula, tetapi sangat penting untuk menjadi programmer C++ yang handal. Latih dengan banyak contoh, dan selalu periksa pointer sebelum digunakan.

18. Object Oriented Programming (OOP)

OOP (Object Oriented Programming) adalah paradigma pemrograman yang memandang program sebagai kumpulan objek yang saling berinteraksi. Setiap objek memiliki data (atribut) dan perilaku (method). C++ adalah salah satu bahasa yang mendukung OOP penuh, bersama dengan Java, C#, dan Python.

🔹 Class dan Object (Cetakan dan Hasil Cetakan)

  • Class adalah cetakan (blueprint) yang mendefinisikan properti dan perilaku. Contoh: class Mobil memiliki atribut merk, warna, dan method klakson().
  • Object adalah instance nyata dari class. Contoh: Mobil Toyota berwarna merah adalah objek dari class Mobil.
class Mobil { public: string merk; string warna; void klakson() { cout << "Beep! Beep!" << endl; } }; int main() { Mobil mobilSaya; mobilSaya.merk = "Toyota"; mobilSaya.warna = "Merah"; mobilSaya.klakson(); // Output: Beep! Beep! return 0; }

🔹 Perbedaan Class dan Struct

Secara teknis, di C++ struct sama seperti class, tetapi secara default anggota struct bersifat public, sedangkan class private. Dalam OOP, kita biasanya menggunakan class karena mendukung enkapsulasi dengan baik.

🔹 Pilar Utama OOP (4 Prinsip Fundamental)

  • Enkapsulasi (Encapsulation) → Menyembunyikan data internal dari luar, hanya diekspos melalui method (getter/setter). Ini melindungi integritas data.
  • Inheritance (Pewarisan) → Class dapat mewarisi atribut dan method dari class lain. Contoh: Kucing mewarisi dari Hewan.
  • Polymorphism (Banyak Bentuk) → Method yang sama dapat memiliki perilaku berbeda di class yang berbeda. Contoh: method suara() pada Kucing mengeluarkan "Meong", pada Anjing "Guk".
  • Abstraksi (Abstraction) → Menyembunyikan detail implementasi dan hanya menampilkan fitur yang penting. Contoh: Anda tahu cara menggunakan cout tanpa tahu detail internalnya.

📘 Contoh Inheritance (Pewarisan)

class Hewan { public: string nama; void makan() { cout << nama << " sedang makan." << endl; } }; class Kucing : public Hewan { // Kucing mewarisi Hewan public: void suara() { cout << "Meong!" << endl; } }; int main() { Kucing k; k.nama = "Kitty"; k.makan(); // dari class Hewan k.suara(); // dari class Kucing sendiri return 0; }

📘 Contoh Polymorphism (Menggunakan Virtual Function)

class Hewan { public: virtual void suara() { // virtual memungkinkan override cout << "Hewan bersuara" << endl; } }; class Kucing : public Hewan { public: void suara() override { // override method dari parent cout << "Meong" << endl; } }; int main() { Hewan *h = new Kucing(); h->suara(); // Output: Meong (bukan "Hewan bersuara") delete h; return 0; }

✅ Keuntungan OOP

  • Modular → Program dipecah menjadi kelas-kelas kecil yang independen.
  • Reusable → Class dapat digunakan ulang di proyek lain.
  • Mudah dipelihara → Perubahan pada satu kelas tidak merusak kelas lain jika antarmuka tetap.
  • Mendekati dunia nyata → Objek lebih alami untuk dimodelkan (mobil, mahasiswa, rekening bank).

💡 Tips Belajar OOP untuk Pemula

  • Mulai dengan class sederhana: buat class Mobil dengan atribut merk dan warna, serta method start().
  • Pahami perbedaan public, private, protected (akses kontrol).
  • Pelajari konstruktor dan destruktor (cara class diinisialisasi dan dibersihkan).
  • Jangan langsung terjun ke polymorphism; kuasai inheritance dulu.
  • OOP adalah topik besar; luangkan waktu dan buat proyek kecil (misal sistem manajemen toko sederhana).

OOP adalah fondasi pengembangan perangkat lunak modern. Setelah Anda menguasai C++ prosedural, OOP akan membuka pintu ke desain sistem yang besar dan kompleks. Selamat belajar!
